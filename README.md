# 🩺 Stroke Prediction System

## Project Overview

This project predicts the risk of stroke using a Machine Learning model and a Streamlit web application.

## Features

- Predicts stroke risk.
- User-friendly interface.
- Built with Python and Streamlit.
- Instant prediction results.

## Technologies Used

- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn

## Project Structure

```
Stroke_Prediction_App/
│
├── app.py
├── stroke_prediction_model.pkl
├── requirements.txt
└── README.md
```

## Installation

1. Clone the repository.

```bash
git clone https://github.com/YourUsername/Stroke_Prediction_App.git
```

2. Install dependencies.

```bash
pip install -r requirements.txt
```

3. Run the application.

```bash
streamlit run app.py
```

## Input Features

- Gender
- Age
- Hypertension
- Heart Disease
- Average Glucose Level
- BMI

## Output

- High Risk of Stroke
- Low Risk of Stroke
