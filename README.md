# Brain Stroke Prediction System

This project predicts whether a patient is at risk of stroke using Machine Learning.

## Features

- Logistic Regression
- Random Forest
- XGBoost
- User-friendly Streamlit interface
- Predicts High Risk or Low Risk of Stroke

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- NumPy

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
